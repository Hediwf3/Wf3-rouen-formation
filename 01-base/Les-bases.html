Les formulaires en HTML

  • La balise FORM permet de creer le formulaire
  • name: permet de donner un nom au formulaire
  • action: Cet attribut est obligatoire et permet de renseigne le script / la page vers laquelle l'utilisateur sera redirigé lors de l'envoie du formulaire. Cette page va recevoir l'ensemble des donées saisie par l'user
  • method:Permet de preciser quelle methode [GET ou POST] doit etree utilisée par le navigateur pour transmettre les données saisies par le user vers la page action















Créer un liens en Html

  • La Balise "a" permet de creer un lien vers une autre ressource. (images, video, pdf, zip,)
  • l'attribu href : permet de definir le chemin vers le fichier a lier.
  • L'attribut target : prend plusier valeur : "_blank" pour ouvrir le lien dans une nouvelle fenetre par exemple
  • l'attribut title : permet de donner le titre au lien lors du survol dudit lien par la souris
